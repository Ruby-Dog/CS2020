A:  
1.  Bandwith
```
1.頻寬
2.可以傳輸的頻率範圍
3.有效頻寬愈高 傳輸速率愈大 
```  
    bps  
```
1.英文全名: bits per second
2.計算一秒傳輸Or處理的位元數
3.是其中一種「位元速率(bit rate)」
```
2.  RFID
```
1.英文全名: Radio Frequency Identifcation
2.可以透過「無線電」識別目標並讀取數據
3.用於 鈔票防偽 身分證 門禁系統
```
3.  NFC
```
1.英文全名: Near-Field communication
2.一種通訊協定 讓兩個裝置在幾公分內進行通訊
3.用於 行動支付
```
4.  cellular network
```
1.蜂巢式網路 
2.又稱「行動網路(mobile network)」
3.如 : LTE  3G 4G 5G
```
